### Files system

Program provide imitation of files tree and operation on leaves with simple console. 

#### Functionalities:
- Add file or directory to tree
- Checking for cycles
- Simple console with commands:
    - exit
    - tree
    - touch
    - mkdir
    - cd
    - ls
    - rm

#### Design patterns:
- Command
- Composite
- Delegation pattern
